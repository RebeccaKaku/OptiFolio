# Akshare 交易摩擦接口调查

**日期**: 2026-06-03
**目的**: 调查 akshare 中有哪些接口可以获取交易摩擦数据（拆并股、手续费、分红、基金限制等）

---

## 1. 交易摩擦分类法

动手前先梳理完整的摩擦类型，再去查 akshare 能覆盖哪些：

```
交易摩擦
├── 1. 显性成本 (直接扣款)
│   ├── 1.1 佣金/手续费 — 券商佣金、交易所费用
│   ├── 1.2 印花税 — A股卖出 0.05%、港股 0.1%（监管固定，无需 API）
│   ├── 1.3 过户费 — A股 0.001%（监管固定）
│   └── 1.4 规费 — 证管费、经手费（监管固定）
│
├── 2. 市场冲击 (隐性成本)
│   ├── 2.1 买卖价差 (bid-ask spread)
│   ├── 2.2 市场冲击成本 — 大单推高/压低价格
│   └── 2.3 滑点 — 下单到成交之间的价格变化
│
├── 3. 流动性约束
│   ├── 3.1 涨跌停限制 — A股 ±10%
│   ├── 3.2 停牌 — 资产重组、重大事项
│   ├── 3.3 T+1 交收 — A股当日买入次日才能卖出（交易规则固定）
│   ├── 3.4 限售股解禁 — IPO 锁定期、定增锁定期
│   └── 3.5 最小交易单位 — A股 100 股（1 手）
│
├── 4. 公司行为 (corporate actions)
│   ├── 4.1 送股 + 转增股本 (stock dividend) — A 股特有，类似 split
│   ├── 4.2 现金分红 (cash dividend)
│   ├── 4.3 配股 (rights offering)
│   ├── 4.4 吸收合并 (merger)
│   ├── 4.5 除权除息 (ex-rights / ex-dividend)
│   ├── 4.6 复权因子 (adjustment factor) — 综合反映所有公司行为
│   └── 4.7 分红税 — 持股 1 年以上免税（监管固定）
│
├── 5. 基金特有摩擦
│   ├── 5.1 申购费 / 认购费 — 前端收费
│   ├── 5.2 赎回费 — 持有越短费率越高
│   ├── 5.3 管理费 + 托管费 — 从净值中每日计提（基金合同固定费率）
│   ├── 5.4 销售服务费 — C 类份额收取
│   ├── 5.5 申购状态 — 开放/暂停/限额
│   ├── 5.6 赎回状态 — 开放/暂停
│   ├── 5.7 下一开放日 — 定期开放式基金
│   ├── 5.8 起购金额 + 单日限额
│   ├── 5.9 封闭期 — 新基金封闭建仓期
│   └── 5.10 清盘 — 规模过小触发
│
├── 6. 跨境摩擦
│   ├── 6.1 QDII 额度限制 — 额度用尽时暂停申购
│   ├── 6.2 跨境税 — 美股股息预扣税 10%（中美协定）
│   └── 6.3 时差 — 海外净值 T+1 公布
│
└── 7. 监管/合规限制
    ├── 7.1 大股东减持限制
    ├── 7.2 外资持股比例上限
    └── 7.3 单只股票持仓上限 — 基金 ≤10%
```

## 2. Akshare 接口实测结果

### 2.1 公司行为 — 分红 / 送股 / 转增 / 配股

#### `stock_fhps_em(date)` — 全市场分红配送（年报视角）

```python
df = ak.stock_fhps_em(date='20251231')
# 3650 条记录, 18 列
```

| 列名 | 含义 |
|------|------|
| 代码 | 股票代码 |
| 名称 | 股票名称 |
| 送转股份-送转总比例 | e.g. 10 表示 10 送/转 10 |
| 送转股份-送股比例 | 送股部分 |
| 送转股份-转股比例 | 转增部分（公积金转增） |
| 现金分红-现金分红比例 | e.g. 1.46 表示每 10 股派 1.46 元 |
| 现金分红-股息率 | 股息率 |
| 每股收益 / 每股净资产 / 每股公积金 / 每股未分配利润 | 财务指标 |
| 净利润同比增长率 / 总股本 | 基本面 |
| 预案公告日 / 股权登记日 / 除权除息日 | **关键日期** |
| 进度 | 实施分配 / 董事会预案通过 / 股东大会通过 |
| 最新公告日期 | — |

**→ 可直接用于自动检测 A 股分红事件，覆盖送股、转增、现金分红。**

#### `stock_fhps_detail_em(symbol)` — 单只股票历史分红

```python
df = ak.stock_fhps_detail_em(symbol='600519')  # 茅台
# 24 条记录（自 2001 年上市以来的每次分红）
```

每行包含：送股比例、转增比例、现金分红金额、除权除息日、股权登记日、进度。

**→ 可用于重建某只股票的完整 corporate action 历史。**

#### `stock_history_dividend_detail(symbol, indicator)` — 分类查询

| indicator | 返回内容 |
|-----------|----------|
| `'分红'` | 送股、转增、派息、进度、除权除息日、股权登记日 |
| `'配股'` | 配股方案、配股价格、基准股本、除权日、缴款起止日 |
| `'拆股'` | 与配股相同（A 股无美股式拆股） |
| `'送股'` | 与配股相同 |

**实测**：万科 A (000002) 有 36 条分红记录和 3 条配股记录。

#### `stock_zh_a_daily(symbol, adjust='qfq-factor')` — 前复权因子

```python
df = ak.stock_zh_a_daily(symbol='sz000002', adjust='qfq-factor')
# [date, qfq_factor]  — 仅在发生公司行为的日期有数据
```

复权因子综合反映了所有公司行为（分红 + 送转 + 配股）对价格的影响：
- `qfq_factor = 1.0` 表示最近一次除权后的基准
- `qfq_factor = 1.05` 表示上一次除权前价格需要乘以 1.05 才能与当前价格可比

**→ 可用来自动计算调整后的价格，无需逐笔追踪每个分红/拆股事件。**

### 2.2 基金费率

#### `fund_fee_em(symbol, indicator)` — 单只基金费率

```python
df = ak.fund_fee_em(symbol='005827', indicator='赎回费率')
```

| indicator 可选值 | 实测结果 |
|------------------|----------|
| `'赎回费率'` | ✅ 返回阶梯费率表 — 如 <7 天 1.5%, >=730 天 0% |
| `'申购费率'` | ❌ 空 (部分基金) |
| `'管理费率'` | ❌ 空 |
| `'托管费率'` | ❌ 空 |
| `'认购费率'` | ❌ 空 |

**间接获取管理费/托管费的方法**：通过 `fund_individual_basic_info_xq` (雪球) 获取基金基本信息，其中包含费率字段。或者从基金合同的固定费率规则推断（股票型 1.5% 管理费 + 0.25% 托管费是行业标准）。

#### `fund_purchase_em()` — 全市场基金申赎状态 🔑

```python
df = ak.fund_purchase_em()
# 26,663 只基金!
```

| 列名 | 含义 | 示例 |
|------|------|------|
| 基金代码 | | 005827 |
| 基金简称 | | 易方达蓝筹精选混合 |
| 基金类型 | | 混合型-偏股 |
| 最新净值/万份收益 | | 1.5959 |
| 最新净值公布时间 | | 06-03 |
| **申购状态** | | **开放申购** / 暂停申购 / 限额申购 |
| **赎回状态** | | **开放赎回** / 暂停赎回 |
| **下一开放日** | | NaT (每日开放) 或具体日期 (定期开放) |
| **起购金额** | | 10.0 元 |
| **单日累计申购限额** | | 50,000 元 (限额申购时) |
| **手续费** | | 0.15% |

**→ 这个接口是基金摩擦数据的关键来源。可直接判断：**
- 能否买入（申购状态）
- 能否卖出（赎回状态）
- 是否处于封闭期（下一开放日）
- 大额申购限制（单日限额）
- 最低投资额（起购金额）
- 申购手续费率

### 2.3 税费 — 无需 API，监管固定

| 费用 | A 股 | 说明 |
|------|------|------|
| 印花税 | 卖出 0.05% | 固定，由财政部规定 |
| 过户费 | 0.001% | 中国结算收取 |
| 分红税 | 持股 >1 年 0% / 1 月-1 年 10% / <1 月 20% | 先进先出计算 |
| 规费 | 约 0.00687% | 证管费 + 经手费 |

**→ 这些应硬编码在 FeeRule 中，不需要 API 查询。**

---

### 2.4 缺失的接口

| 摩擦类型 | Akshare 是否支持 | 替代方案 |
|----------|------------------|----------|
| 美股拆股 (stock split) | ❌ A 股无此概念 | 送股/转增代替；用 `yfinance.actions` 查美股 |
| 吸收合并 (merger) | ❌ 无专用接口 | `stock_fhps_em` 可能不覆盖 |
| **停牌** | ❌ 无结构化接口 | `news_trade_notify_suspend_baidu` (新闻而非数据) |
| **限售股解禁** | ❌ 无接口 | 需要查东方财富限售股页面 |
| 涨跌停 | ❌ 无接口 | 交易规则固定 (±10% 主板, ±20% 科创/创业) |
| 买卖价差 | ❌ 需要 tick 数据 | 可通过 `stock_zh_a_tick_tx` 获取 |
| QDII 额度 | ❌ 无接口 | 外汇管理局公布，非 akshare 范围 |
| 基金管理费/托管费 | 🟡 `fund_fee_em` 返回空 | 用 `fund_individual_basic_info_xq` 或基金合同固定费率 |
| **港股分红/拆股** | ✅ `stock_hk_fhps_detail_ths` | — |

### 3.3 雪球接口 (Xueqiu) 已解决 ✅

通过 `fund_individual_detail_info_xq` + `fund_individual_basic_info_xq` 获得：
- 管理费 (management fee) — e.g. 1.20%/年
- 托管费 (custody fee) — e.g. 0.20%/年
- 申购费阶梯 — 按购买金额分档
- 赎回费阶梯 — 按持有天数分档
- 基金全称、成立时间、最新规模、基金经理、投资策略

雪球的费用名称与 East Money 不同，采用 Unicode 匹配同时兼容两套命名。
