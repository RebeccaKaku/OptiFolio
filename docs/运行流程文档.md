# FM金融管理系统 - 运行流程文档

## 项目概述

FM是一个基于Python的金融管理系统，提供资产配置、投资组合管理、风险分析和可视化仪表板功能。系统采用模块化设计，支持多种资产类型（股票、基金、货币等）的数据获取、分析和优化。

## 技术架构

### 核心技术栈
- **后端框架**: Flask + Streamlit
- **数据获取**: AkShare (中国金融数据) + yfinance (美股数据)
- **数据处理**: Pandas + NumPy
- **数学优化**: CVXPY (凸优化)
- **可视化**: Plotly + Streamlit
- **缓存**: cachetools
- **配置管理**: YAML

```python
def import_asset(symbol, asset_type=None, refresh=False):
    # 1. 标准化资产代码
    normalized_symbol = normalize_symbol(symbol, asset_type)
    
    # 2. 智能识别资产类型（如果未指定）
    if not asset_type:
        asset_type = infer_asset_type(normalized_symbol)
    
    # 3. 创建资产定义
    asset_def = AssetDefinition(
        symbol=normalized_symbol,
        asset_type=asset_type,
        name=None,  # 稍后从API获取
        currency=None  # 稍后检测
    )
    
    # 4. 从数据源获取详细信息
    api_data = fetch_asset_info(normalized_symbol, asset_type)
    
    # 5. 更新资产信息
    if api_data:
        asset_def.update_from_api(api_data)
    
    # 6. 智能币种检测
    if not asset_def.currency or asset_def.currency == 'CNY':
        detected_currency = detect_currency(asset_def.name)
        if detected_currency != 'CNY':
            asset_def.currency = detected_currency
    
    # 7. 注册到资产注册表
    registry.register_asset(asset_def)
    
    # 8. 保存配置
    registry.save_config()
    
    return asset_def
```

```python
def fetch_market_data(assets, start_date, end_date):
    # 1. 初始化数据加载器
    data_loader = DataLoader(config)
    
    # 2. 遍历所有资产
    for asset in assets:
        symbol = asset['symbol']
        asset_type = asset['type']
        
        # 3. 获取对应的Fetcher
        fetcher = factory.get_fetcher(asset_type)
        
        # 4. 获取原始数据
        raw_data = fetcher.fetch(symbol, start_date, end_date)
        
        # 5. 数据验证
        if validator.validate(raw_data):
            # 6. 数据清洗
            cleaned_data = processor.clean(raw_data)
            
            # 7. 存储原始数据
            storage.save_raw(symbol, cleaned_data)
            
            # 8. 添加到缓冲区
            raw_buffer[symbol] = cleaned_data
        
        # 9. 防封控延迟
        time.sleep(0.5)
    
    # 10. 数据对齐处理
    aligned_data = processor.align_and_clean(raw_buffer, start_date, end_date)
    
    # 11. 保存处理后的数据
    storage.save_processed(aligned_data)
    
    return aligned_data
```

### 5. 优化算法流程

```python
# 伪代码 - 投资组合优化
def optimize_portfolio(expected_returns, covariance_matrix):
    # 1. 定义优化变量
    n_assets = len(expected_returns)
    weights = cp.Variable(n_assets)
    
    # 2. 定义目标函数
    # 效用 = 预期收益 - 风险厌恶系数 * 风险
    portfolio_return = weights @ expected_returns
    portfolio_risk = cp.quad_form(weights, covariance_matrix)
    objective = cp.Maximize(portfolio_return - risk_aversion * portfolio_risk)
    
    # 3. 定义约束条件
    constraints = [
        cp.sum(weights) == 1,           # 满仓约束
        weights >= 0,                   # 做多约束
        weights <= max_position_size    # 分散化约束
    ]
    
    # 4. 求解优化问题
    problem = cp.Problem(objective, constraints)
    problem.solve()
    
    # 5. 处理结果
    optimal_weights = pd.Series(weights.value, index=expected_returns.index)
    optimal_weights[optimal_weights < 0.0001] = 0  # 清理微小权重
    optimal_weights = optimal_weights / optimal_weights.sum()  # 重新归一化
    
    return optimal_weights
```

## 运行步骤

### 步骤1: 环境准备

```bash
# 1. 安装依赖
pip install -r requirements.txt
# 或
pip install -e .

# 2. 验证安装
python -c "import streamlit; import akshare; import yfinance; print('All packages installed successfully')"
```

### 步骤2: 启动应用

```bash
# 启动Streamlit应用
streamlit run app.py
```

### 步骤3: 系统初始化

应用启动后会自动：
1. 加载配置文件
2. 初始化API服务
3. 检查系统状态
4. 在浏览器中打开 `http://localhost:8501`

### 步骤4: 资产导入

在"资产导入"页面：
1. 输入资产代码（如：sh000001, AAPL, EUR/USD）
2. 选择资产类型（可选，系统会自动识别）
3. 点击"导入资产"按钮
4. 系统将自动获取资产信息并保存到注册表

### 步骤5: 组合管理

在"组合管理"页面：
1. 查看当前持仓和价值
2. 添加新持仓或移除现有持仓
3. 更新现金余额
4. 查看组合分析和再平衡建议

### 步骤6: 数据分析

在"分析工具"页面：
1. 查看组合性能图表
2. 分析风险指标
3. 比较不同资产的表现
4. 进行压力测试

## 故障排除

### 常见问题

1. **数据获取失败**
   - 检查网络连接
   - 验证API服务状态
   - 查看日志文件

2. **优化求解失败**
   - 检查数据完整性
   - 验证约束条件
   - 调整风险参数

3. **性能问题**
   - 清理缓存
   - 减少同时分析的资产数量
   - 使用更快的求解器
